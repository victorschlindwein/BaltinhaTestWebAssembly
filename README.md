# BaltinhaTest

## ATENÇÃO
Esse projeto trata-se apenas de um front-end criado em Blazor Web Assembly.  


### Home
Ele se conecta com uma outra aplicação criada pelo Balta, disponível nesse repositório: [encontro-premium-2023-05-04](https://github.com/balta-io/encontro-premium-2023-05-04/tree/main/01-api).  

Esse projeto foi criado para servir o back-end contido dentro da pasta "01-api".


### Cities
Essa aba se conecta localmente, através do arquivo ``Configuration.cs`` com a api desenvolvida no [Desafio do Balta](https://github.com/victorschlindwein/DesafioBalta) e faz o uso dela como back-end.

 <strong>OBS: Para utilizar a API do IBGE likada como modelo, você DEVE habilitar o CORS dentro do ``Program.cs`` disponibilizado no repositório acima. </strong>
